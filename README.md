# vlessok

Минималистичный и мощный VPN-клиент для Windows на базе **Tauri** и движка **sing-box**.

Протокол: **VLESS + Reality + Vision**  
Режим работы: Системный **TUN**-туннель (полный перехват трафика на сетевом уровне).

---

## 📥 Быстрая установка (Для пользователей)

Если вы хотите просто использовать приложение, вам **не нужно** ничего компилировать или устанавливать сторонние инструменты:

1. Перейдите на страницу **[Релизов (Releases)](https://github.com/Shershshsh/vlessok/releases)**.
2. В самом верхнем релизе (последняя версия) в блоке **Assets** скачайте файл **`vlessok_1.0.0_x64-setup.exe`** (это автоматический установщик).
3. Запустите скачанный файл и следуйте инструкциям установщика.
4. После установки запустите приложение через ярлык на Рабочем столе.
   * *Примечание:* При подключении приложение автоматически запросит права администратора (UAC) — это необходимо для создания виртуальной сетевой карты TUN.

## ✨ Ключевые возможности (v1.0.0)

* **Режим TUN (Системный VPN):** Полноценное виртуальное сетевое устройство (`vlessok-tun`). Никаких проблем с программами, не поддерживающими системный прокси.
* **Гибкая маршрутизация (Split Tunneling):**
  * **Глобальный режим:** Весь трафик идет через прокси.
  * **Режим правил:** Выборочное проксирование трафика. Поддерживает:
    * **Домены:** Удобное добавление сайтов (ссылки автоматически нормализуются до чистых доменов).
    * **GEOIP / GEOSITE:** Стандартные правила маршрутизации (например, `geosite:google`, `geosite:netflix`).
    * **Готовые пресеты правил:**
      * **Russia Pack** (`inside-raw.lst`) — автоматический обход блокировок сайтов в РФ (автоматически скачивается и обновляется).
      * **Telegram Combo** — проксирование доменов и процесса Telegram.
      * **Discord Combo** — проксирование доменов и процесса Discord.
    * **Процессы (App-based VPN):** Вы можете проксировать только отдельные приложения (например, `browser.exe`, `telegram.exe`). Доступен **Диспетчер процессов** прямо в UI, позволяющий выбрать запущенную программу из списка с отображением её иконки.
* **Стабильность и безопасность:**
  * **Защита от утечек DNS (DNS Leak Protection):** Тонкая настройка DNS-клиента Windows через реестр и PowerShell.
  * **Гарантия завершения процессов (Job Objects):** sing-box привязывается к UI через Windows Job Objects. При закрытии, аварийном завершении или зависании графической оболочки сетевой движок гарантированно завершит работу, не оставляя систему без интернета.
  * **Сброс сетевых настроек:** Специальная кнопка восстановления сетевых интерфейсов и реестра DNS на случай сбоев.
  * **UAC-интеграция:** Приложение автоматически определяет права доступа и запрашивает перезапуск от имени администратора при необходимости.

## 🛠️ Системные требования

### Для работы готовой сборки (пользователям):
* Windows 10 / 11 (64-bit).
* Запуск от имени администратора (для создания TUN-интерфейса).

### Для разработки и сборки из исходников:
| Инструмент | Минимальная версия | Как проверить |
|---|---|---|
| **Rust** | 1.77+ | `rustc --version` |
| **Node.js** | 18+ | `node --version` |
| **Visual Studio Build Tools** | 2022 | Требуется компонент "C++ build tools" и Windows SDK |

## 🚀 Быстрый старт для разработчиков

### 1. Скачайте зависимости ядра
Для работы бэкенда в папке `src-tauri/binaries/` должны находиться:
1. `sing-box.exe` (версии 1.10+).
2. `wintun.dll` (драйвер TUN-интерфейса).

*Они занесены в `.gitignore` и не попадают в репозиторий.*

### 2. Установите зависимости Node.js
```bash
npm install
```

### 3. Запустите в dev-режиме
```bash
npm run tauri dev
```
*При первом запуске компиляция Rust-зависимостей займет несколько минут. Последующие запуски происходят мгновенно.*

## 📦 Сборка дистрибутива

Чтобы собрать готовый установщик для распространения, выполните:
```bash
npm run tauri build
```

После завершения сборки файлы появятся в следующих папках:
* **Standalone EXE:** `src-tauri/target/release/vlessok.exe`
* **Инсталлятор NSIS (.exe):** `src-tauri/target/release/bundle/nsis/vlessok_1.0.0_x64-setup.exe`
* **Инсталлятор WiX (.msi):** `src-tauri/target/release/bundle/msi/vlessok_1.0.0_x64_en-US.msi`

## 📁 Структура проекта

```
vlessok/
├── src/                    # Фронтенд (HTML, CSS, JS)
│   ├── index.html          # Разметка UI
│   ├── styles.css          # Дизайн, анимации, адаптивная верстка
│   └── main.js             # Логика UI, интеграция с Tauri API
├── src-tauri/              # Бэкенд (Rust)
│   ├── binaries/           # Бинарные файлы ядра (sing-box, wintun)
│   ├── src/
│   │   ├── main.rs         # Точка входа Tauri-приложения
│   │   ├── lib.rs          # Мост Tauri-команд, процессы, интеграция WinAPI
│   │   ├── config.rs       # Генератор JSON-конфигураций для sing-box
│   │   ├── platform.rs     # Утилиты Windows (Job Objects, UAC проверка)
│   │   ├── routing.rs      # Структура и логика правил маршрутизации
│   │   ├── russia_pack.rs  # Скачивание и обновление списков обхода блокировок
│   │   └── singbox.rs      # Менеджер жизненного цикла процесса sing-box
│   ├── Cargo.toml          # Зависимости Rust
│   └── tauri.conf.json     # Настройки сборки и манифеста Tauri
├── package.json            # Зависимости npm и скрипты запуска
└── README.md               # Документация проекта
```
