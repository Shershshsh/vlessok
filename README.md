# vlessok

Минималистичный VPN-клиент для Windows.  
Протокол: **VLESS + Reality + Vision**  
Режим работы: **TUN** (полный перехват трафика)

---

## 📋 Требования

Перед началом работы убедись, что установлены:

| Инструмент | Минимальная версия | Как проверить |
|---|---|---|
| **Rust** | 1.70+ | `rustc --version` |
| **Node.js** | 18+ | `node --version` |
| **npm** | 8+ | `npm --version` |

### Установка Rust (если не установлен):
```
https://rustup.rs/
```
Скачай и запусти `rustup-init.exe`, следуй инструкциям.

### Установка Node.js (если не установлен):
```
https://nodejs.org/
```
Скачай LTS-версию и установи.

## 🚀 Быстрый старт

### 1. Установи зависимости
```bash
cd путь/к/папке/vlessok
npm install
```

### 2. Запусти в dev-режиме
```bash
npm run tauri dev
```

**Первый запуск** может занять 3-10 минут — Rust компилирует все зависимости.  
Последующие запуски будут быстрыми (несколько секунд).

### 3. Что ты должен увидеть

Откроется окно приложения с:
- Заголовком **"vlessok"**
- Подзаголовком **"VPN-клиент · VLESS + Reality"**
- Кнопкой **"🔌 Проверить связь с бэкендом"**

Нажми кнопку → должно появиться зелёное сообщение:  
**"✅ Hello from Rust! Бэкенд работает."**


## 🐛 Типичные проблемы

### ❌ `error: linker 'link.exe' not found`
**Решение:** Установи [Visual Studio Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/).  
При установке выбери "C++ build tools" и "Windows SDK".

### ❌ `npm run tauri dev` ничего не делает
**Решение:** Проверь что `npm install` прошёл без ошибок.

### ❌ Окно открывается, но кнопка не работает
**Решение:** Открой консоль разработчика (Ctrl+Shift+I в окне приложения) и посмотри ошибки в Console.

### ❌ `error[E0463]: can't find crate`
**Решение:** Удали папку `src-tauri/target` и запусти заново:
```bash
rm -r src-tauri/target
npm run tauri dev
```
